# SQL Notes

This capstone does not currently require BigQuery SQL.

The analysis is fully reproducible with the raw Kaggle CSV, Python scripts, and saved output files in this project folder. The college brief recommends SQL when joining or aggregating large datasets, but the selected OSMI dataset is a single survey CSV, so Python was the simpler and more transparent choice.

If this project is later moved into BigQuery, save every query in this folder so the work is not tied only to a temporary BigQuery account.
